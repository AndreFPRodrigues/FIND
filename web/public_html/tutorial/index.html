Alpha test group

First join our alpha test group. To do so head on over to google groups sign in onto your google account and click the join button.

Android Applications: Victim and Rescuer

You can only download and install one of the applications. Each app comes bundled with the FIND Service application. To be able to download one of the applications please register as a alpha tester in as described bellow.

Find Victim

To download and install the application please follow the link Victim app and register as an alpha tester. Then follow the link to download the application from the play store. In the application descriptions you can find out more about about the Find Victim application. FIND Service is bundled with FIND Victim.

Instructions

  • First start FIND Service application with a WI-FI internet connection to register the device;
  • By default you will receive an pop up notification whenever a simulation is created;
  • You can then decide to join the simulation or not;
  • When registered in a simulation FIND Service will automatically start and stop on conclusion;
  • When in a simulation Find Victim allows you to send messages;
  • Communication is accomplished either across devices or through Wi-Fi internet connection;
  • Launching FIND Victim will start the FIND Service, to terminate it, go to FIND Service;

Disclaimer

  • All information is currently disseminated across devices and to our webserver;
  • We only store anonymous location data of registered users locally during the simulation period. Upon conclusion the data is deleted from all devices and only gathered in our webserver;
  • We do not guarantee compatibility with all devices;
